
# MyMoviesApp

A brief description of what this project does and who it's for


## Install

Clone repository:
```js
git clone https://github.com/alexrequeno/MyMoviesApp.git
```

Install node packages:
```js
npm install
```

## Run

Run on Android:
```js
npx react-native run-android
```

Run on iOS:
```js
npx react-native run-ios
```
